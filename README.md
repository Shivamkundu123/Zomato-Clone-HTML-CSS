# Zomato-Clone-HTML-CSS
A responsive Zomato landing page clone built using HTML5, CSS3, and Normalize.css. The project replicates the basic UI layout including hero section, food categories, and search bar. It focuses on clean design, Flexbox layout, and cross-browser compatibility for consistent styling
Zomato Clone is a front-end web development project created using pure HTML5, CSS3, and Normalize.css for cross-browser consistency.

This project recreates the user interface of a food delivery website similar to Zomato, focusing on layout structure, responsive design, and clean styling.

🚀 Features

Hero section with background image

Search input and call-to-action button

Food category sections (Snacks, Breakfast, Dinner)

Responsive Flexbox layout

Mobile-friendly design

Cross-browser styling using Normalize.css

Clean and organized folder structure

🛠 Technologies Used

HTML5

CSS3 (Flexbox)

Normalize.css

Google Fonts (Montserrat)

🎯 Purpose of the Project

This project was built to practice:

Web page layout design

Flexbox positioning

Responsive web design

Clean UI structure

Real-world website cloning practice
